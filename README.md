## 목표

이 Study 문서의 목표는 블록체인의 기본적인 개념을 이해하고, 그 개념을 눈에보이는 실체로 구현하는 것을 목표로 합니다. 실제 상용화 가능한 용도까지는 도달하지 못할 수 있으니 참고 부탁드립니다.
